# Getting started
*  Add Node Packages

        npm iit

* Unit Testing

Install Jest using yarn:

        yarn add --dev jest 

Or npm:

        npm install --save-dev jest

* Sass
  
Install Sass with npm

    npm install -g sass
Then watch sass with this command 

    sass --watch input.scss output.css
